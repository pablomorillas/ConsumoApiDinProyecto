# Detalle de Edificios

Esta sección documenta el componente \`DetalleEdificios\`, utilizado para visualizar la información detallada de los inmuebles.

A continuación se muestran los ejemplos interactivos (Historias) renderizados directamente desde Storybook.

:::info
Para ver estos ejemplos correctamente, asegúrate de que tu Storybook esté ejecutándose en \`http://localhost:6006\`.
:::
