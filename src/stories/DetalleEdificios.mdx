# Detalle de Edificios

El componente **DetalleEdificios** es la vista principal para mostrar la información completa de un inmueble. 
Está diseñado para ser utilizado tanto en la navegación principal (cargando datos dinámicamente vía URL) como de forma aislada (recibiendo datos directos vía props).

## Propósito

Este componente sirve para:
- Mostrar la imagen destacada del edificio.
- Listar características clave: Nombre, Precio, Categoría y Descripción detallada.
- Proveer navegación de retorno al listado principal.

## Uso Básico

El componente se integra generalmente con el `Router` de la aplicación. Por defecto, intenta leer el parámetro `:index` de la URL para buscar el edificio en el array de datos global.

## Modo "Playground" (Interactivo)

Para propósitos de desarrollo y testeo, el componente admite la inyección directa de datos a través de la prop `edificioData`. 
Esto es útil para visualizar cómo se renderizaría el componente con datos que aún no existen en la base de datos o para probar casos extremos (textos muy largos, precios altos, etc.).

## Notas de Implementación

- **Dependencias**: Requiere `react-router-dom` para el botón de "Volver" (`useNavigate`) y la lectura de parámetros (`useParams`).
- **Validación**: Si el índice no existe o los datos no se proveen, mostrará un mensaje de error amigable.
